# Machine Learning 

This repository contains work on projects for the Holberton Machine Learning Specialization, focused on several key modules starting with fundamental math concepts, before diving into major machine learning branches such as supervised learning and unsupervised learning models and developing
machine learning pipelines for fast and efficient execution. 



- [Math](https://github.com/evisp/holbertonschool-machine_learning/tree/main/math) includes fundamentals of math for machine learning. From matrices to calculus, this module equips you with the mathematical tools necessary to understand algorithms and models deeply.
